\subsection{levels2osc}\label{sec:levels2osc}\index{levels2osc}\index{level meter}

\begin{tscattributes}
\indattr{pattern} & Level source object name               \\
\indattr{url}     & Target OSC url                         \\
\indattr{ttl}     & Time to live of OSC multicast messages \\
\end{tscattributes}

This module reads the level meters of the specified objects and sends
their values as OSC data and LSL streams.
%
A potential application is data logging of levels.
