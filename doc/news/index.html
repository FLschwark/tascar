TASCAR

Toolbox for acoustic scene creation and rendering

TASCAR is an open-source software platform for real-time interactive virtual acoustic environment rendering in laboratory environments, licensed under GPL-2.0. The source code is hosted at https://github.com/gisogrimm/tascar/. Binary packages and commercial support is available from Hörtech gGmbH.
